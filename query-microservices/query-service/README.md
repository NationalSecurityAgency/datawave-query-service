# Query Service

[![Apache License][li]][ll] ![Build Status](https://github.com/NationalSecurityAgency/datawave/workflows/Tests/badge.svg)

The query api service is a DATAWAVE microservice that provides
query capabilities.

### Query Context

*https://host:port/query/v1/*
